# Business Card - freeCodeCamp Project

A modern and stylish digital business card website built as part of the freeCodeCamp "Responsive Web Design" certification. This project focuses on CSS layout techniques, typography, and precise element positioning.

## 🚀 Technologies Used
* **HTML5:** Semantic markup structure.
* **CSS3:** Custom properties (variables), `font-family` (Arial, sans-serif), text decorations, and container styling.

## 📝 Features
* Clean and minimalist card design centered on the screen.
* Styled text with custom typography fallback settings.
* Customized hyperlinks with removed underlines (`text-decoration: none`).
* Fully responsive and viewable across modern web browsers.
